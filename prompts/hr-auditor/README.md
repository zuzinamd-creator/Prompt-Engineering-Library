## 🚀 Основные возможности

1. **Business Decoding:** Анализ вакансии "между строк" для выявления истинных болей заказчика.
2. **Technical Audit:** Проверка стека кандидата на соответствие требованиям (Hard Threshold 75%).
3. **Resource Planning:** Оценка времени на тестовое задание и планирование бюджета на API.
4. **Strategic Outreach:** Генерация текста отклика в стиле *Human-to-Human* с фокусом на решение бизнес-задач.

## 🧪 Тест-кейсы

### ✅ Кейс №1: Идеальное совпадение
* **Вход:** Вакансия "AI Developer (n8n, Python)", Стек кандидата: LLM, No-code.
* **Ожидаемый выход:** Высокий % матчинга, акцент на автоматизацию и быструю интеграцию.

### ⚠️ Кейс №2: Недостаточная квалификация
* **Вход:** Вакансия "Senior Data Scientist (PyTorch, C++)", Стек кандидата: LLM, No-code.
* **Ожидаемый выход:** Вердикт **"Не рекомендовано"** (матчинг < 30%).

### 📚 Кейс №3: Потенциал роста
* **Вход:** Вакансия "Prompt Engineer", Стек кандидата: LLM, RAG.
* **Ожидаемый выход:** План обучения по недостающим глубоким аспектам RAG + персонализированный отклик.

## 🛠 Инструкция по использованию

Для запуска инструмента скопируйте содержимое файла `prompt.md` в вашу LLM (GPT-4o, Claude 3.5 Sonnet) и заполните переменные:
- `{{candidate_name}}`
- `{{candidate_stack}}`
- `{{job_description}}`
