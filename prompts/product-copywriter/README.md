**AI Product Copywriter** — интеллектуальный инструмент для генерации продающего контента для маркетплейсов (Ozon, Wildberries, Yandex Market). В основе лежит метод самопроверки фактов, что гарантирует 100% соответствие описания реальным характеристикам товара.

---

## 🚀 Ключевые особенности

* **Chain-of-Verification (CoVe):** Модель сама задает себе проверочные вопросы по характеристикам товара перед выдачей финального текста.
* **SEO-оптимизация:** Автоматический подбор ключевых слов на основе описания.
* **Структура "под ключ":** Генерация названия, описания, преимуществ и таблицы характеристик в одном ответе.

## 🧪 Тест-кейс
* **Вход:** Название "Ультразвуковой увлажнитель воздуха", Характеристики "Объем 5л, 3 режима, бесшумный, подсветка".
* **Ожидаемый выход:** Чек-лист верификации (проверка объема и режимов) + Структурированная карточка товара с выделенными преимуществами.
