## 🏗 Принципы разработки

Все промпты в этой библиотеке спроектированы с учетом следующих стандартов:
1. **Zero-Hallucination:** Жесткие ограничения на выдумывание данных.
2. **Business-First:** Фокус на решение конкретных бизнес-задач (KPI, ФОТ, время).
3. **Modularity:** Использование переменных `{{variable}}` для легкой интеграции в любой рабочий процесс.
4. **Structured Output:** Обязательное наличие JSON-блоков для удобной выгрузки в CRM или базы данных.

---

## 🚀 Как использовать
1. Перейдите в папку интересующего вас инструмента.
2. Ознакомьтесь с инструкцией в локальном `README.md`.
3. Скопируйте содержимое файла `prompt.md` в вашу LLM (GPT-4, Claude).
4. Заполните входные переменные в разделе `INPUT DATA`.
